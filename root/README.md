# Root Assets

`root/` 目录用于存放静态页面资源、样式文件和上传文件目录。

## 页面入口

- `index.html`：统一入口页
- `login.html`：登录页
- `register.html`：注册页
- `login-error.html`：登录失败页
- `register-error.html`：注册失败页
- `welcome.html`：登录后欢迎页
- `files.html`：文件管理页

## 兼容别名

- `judge.html` -> `index.html`
- `log.html` -> `login.html`
- `logError.html` -> `login-error.html`
- `registerError.html` -> `register-error.html`

## 历史 Action 标志位

- `0`：注册
- `1`：登录
- `2`：登录检测
- `3`：注册检测
- `5`：历史图片页
- `6`：历史视频页
- `7`：历史关注页
